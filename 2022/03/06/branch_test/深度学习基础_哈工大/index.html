深度学习基础_哈工大

模型数据集的划分

可以使用sklearn 中的工具帮助划分

  1. Hold-Out Method：步骤简单，但是模型评估水平与数据拆分有很大关系不够稳定

  2. K折交叉验证：将数据集分成K份，每次取K-1 训练 1 测试，会得到 K 个不同的评价取均值作为结果。

  3. 分层抽样：根据标签的比例来进行抽样。

    机器学习模型的评估

    分类问题性能评价方式

  4. accuracy, precision, recall, P-R 曲线 AUC。

  5. $$F_{\beta}-\text { Score }=\frac{\left(1+\beta^{2}\right) * \text { precision } * \text { recall }}{\left(\beta^{2} * \text { precision }+\text { recall }\right)}$$其中$$\beta$$大于1 更多关注recall 小于1 更多关注precision

  6. ROC曲线

  7. ROC-AUC

    回归问题性能评价方式

    常用方式MAE, MSE, Log lOSS or cross-entropy loss, R方值，确定系数( r2_cross )

(对于交叉熵或者称为以最大似然估计得到的loss 需要后续自己推导一下)

一致性评价指标

皮尔森( pearson )相关系数， Cohen’s kapaa 相关系数, Fleiss’ kappa

人工智能/深度学习芯片

CPU, GPU, FPGA, ASIC
